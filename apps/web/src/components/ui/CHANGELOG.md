# Changelog

Toutes les modifications notables de cette bibliothèque de composants seront documentées dans ce fichier.

## [1.0.0] - 2025-01-XX

### Ajouté
- **CommandPalette** - Palette de commandes moderne avec raccourci ⌘K
- **MultiSelect** - Sélection multiple avec tags et recherche
- **RichTextEditor** - Éditeur de texte riche avec barre d'outils
- **Presets de thème** - 5 presets (default, modern, corporate, vibrant, minimal)
- **Variables de thème avancées** - Polices multiples, couleurs de texte personnalisables
- **Documentation API** complète
- **Exemples SaaS** - Dashboard, Settings, Onboarding
- **Configuration Storybook** fonctionnelle

### Amélioré
- **Migration vers le système de thème** - Toast, KanbanBoard, Calendar, CRUDModal
- **ThemeManager** - Interface améliorée avec presets
- **Documentation** - README et guides complets

### Corrigé
- Couleurs hardcodées remplacées par variables CSS du thème
- Cohérence du design améliorée

---

## [0.9.0] - 2024-12-XX

### Ajouté
- Composants de base (Button, Input, Card, etc.)
- Composants de données (Table, DataTable, StatsCard)
- Composants de feedback (Alert, Modal, Toast)
- Composants de navigation (Tabs, Accordion, Sidebar)
- Système de thème de base
- Support dark mode
