# Storybook

Storybook est configuré pour ce projet. Pour l'utiliser :

## Installation

```bash
pnpm install
```

## Utilisation

```bash
# Démarrer Storybook
pnpm storybook

# Build Storybook pour production
pnpm build-storybook
```

Storybook sera accessible sur `http://localhost:6006`

## Créer une nouvelle Story

Créez un fichier `.stories.tsx` à côté de votre composant. Voir `Button.stories.tsx` et `Input.stories.tsx` pour des exemples.
